# DnD Wanted Poster Generator

A full-stack AI-powered web app that generates dramatic fantasy wanted posters for DnD campaigns. Built with Vue.js, TypeScript, and FastAPI.

## Features
- Generates a fully styled wanted poster from character details
- AI-generated portrait using DALL-E 3 when no photo is uploaded
- Optional photo upload to use as the mugshot
- Dramatic old English poster text, alias, bounty, threat level, and caution generated by AI
- Clean old paper style poster design with Generate Another button

## Tech Stack
- Frontend: Vue.js, TypeScript, Vite
- Backend: FastAPI, Python
- AI: OpenAI GPT and DALL-E 3

## Prerequisites
- Python 3.10+
- Node.js 20+
- OpenAI API key

## Setup

### Backend
1. Navigate to the backend folder:
   - cd backend
2. Create and activate a virtual environment:
   - python -m venv venv
   - .\venv\Scripts\activate
3. Install dependencies:
   - pip install fastapi uvicorn python-dotenv openai
4. Create a .env file in the backend folder:
   - OPENAI_API_KEY=your_key_here
5. Start the backend:
   - uvicorn main:app --reload

### Frontend
1. Navigate to the project root in a separate terminal
2. Install dependencies:
   - npm install
3. Start the frontend:
   - npm run dev
4. Open http://localhost:5173 in your browser

## How to Use
1. Fill in your character's name, race, class, description, crimes, and last known location
2. Optionally upload a photo to use as your mugshot
3. Click Generate Wanted Poster
4. The AI generates a complete wanted poster with a DALL-E portrait if no photo was uploaded
5. Hit Generate Another to create a new poster

## Notes
- Both the frontend and backend must be running at the same time
- DALL-E portrait generation takes a few extra seconds
- Never commit your .env file, your API key should stay local
